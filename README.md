## HtmlParser

javascirpt 로 구현한 HtmlParser입니다.
코어 함수는 tokenizer와 treeBuilder 파일에 들어있으며 App파일을 통해서 실행됩니다.

### 실행방법

node.js v18.0.0 이상 버전을 설치해주세요.

터미널에 아래 명령어를 입력해 App.js 파일을 실행해주세요

```
node App.js
```

다른 html을 적용해보고 싶다면 App.js의 html변수를 수정해주세요 다만 모든 예외를 구현한것은 아니기 때문에 일부 html은 정상적으로 동작하지 않을 수 있습니다.
